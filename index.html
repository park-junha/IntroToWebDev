Intro to Web Development

Why develop websites / web applications?

The Web is an accessible output of information with a lot of demand in the job market. Many software developers / engineers start out with web development to break into the tech industry!

How do you develop websites?

There are a lot of tools and technologies out there, but the three fundamental components of web development are HTML, CSS, and JavaScript.

  • HTML defines the structure and content of a website.
  • CSS defines how a website should be styled.
  • JavaScript makes a website more interactive.

HTML

Hypertext Markup Language structures websites. You can use tags like <div>, <p>, <h1>, etc. to define the content of a website.

Example

The following HTML will write a paragraph with some bolded words:

<p>I am <b>not</b> the imposter!</p>

I am not the imposter!

CSS

Cascading Style Sheets styles websites.

Example

The following CSS will make the following h4 tag element red:

<h4 style="color: #ff0000">I am feeling red</h4>

I am feeling red

You can also apply these styles in a separate CSS file to make all elements apply the same styles:

h5 { color: #ff0000; }
I am also feeling red
As am I...

NOTE: To apply global styles in a separate CSS file, you must define the stylesheet in the <head> tag of your HTML document.

<head> <link rel="stylesheet" href="styles.css"></link> </head>

JavaScript

JavaScript can program interactive behavior in a website. You can use it to change HTML content, CSS, etc.

Example

The snippet below defines a function that changes the text of the HTML element with the id funny-output.

var funnyCount = 0; function getFunnyCountText() { var postfix; if (funnyCount === 1) { postfix = " time."; } else { postfix = " times."; } return "You clicked the funny button " + funnyCount + postfix; } function funnyScript() { funnyCount++; document.getElementById("funny-output").innerHTML = getFunnyCountText(); }

The following HTML defines a button that calls the JavaScript function funnyScript() when clicked.

<button id="funny-button" onclick="funnyScript()">Click me</button> <div id="funny-output">Nothing to see here</div>
Nothing to see here

NOTE: To apply JavaScript from a separate JavaScript file, you must define it in the <head> tag of your HTML document.

<head> <script type="text/javascript" src="funny.js"></script> </head>
